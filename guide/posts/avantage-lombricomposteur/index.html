Quels sont les avantages d’un lombricomposteur?

Les vers de terre, ou lombrics, sont des animaux fascinants. Ils ont la capacité de transformer la matière organique en compost. Cette activité est naturelle et se déroule dans le sol. Les vers de terre sont présents partout dans le monde. En France, ils sont appelés « ver de fumier ». Ils sont utilisés depuis des siècles pour fertiliser les sols. Le lombricompostage est une technique de recyclage de la matière organique qui permet de réduire le volume de vos déchets alimentaires. Le lombricompostage est un processus biologique naturel. Il ne nécessite aucune intervention humaine.

Que mettre dans un lombricomposteur ?

Les vers de terre ont besoin de matière organique pour se nourrir. Vous pouvez donc mettre dans votre lombricomposteur tous les déchets alimentaires que vous souhaitez : épluchures de fruits et légumes, restes de repas, coquilles d’œufs, marc de café, thé, pain, etc.

Les déchets de cuisine peuvent être mis directement dans le lombricomposteur.

Vous pouvez également y mettre les déchets de jardin : tonte de gazon, feuilles mortes, fleurs fanées, etc. En revanche, il est déconseillé de mettre dans le lombricomposteur : les déchets de viande, poisson, fromage, produits laitiers, huile, etc. Les déchets de cuisine doivent être coupés en petits morceaux (pas de pelure ni de peau). Les déchets de jardin doivent être broyés.

Comment entretenir son lombricomposteur?

L’entretien du lombricomposteur est très simple. Il suffit de remuer le contenu du bac une à deux fois par semaine. Pour que les vers de terre puissent se reproduire, il faut les nourrir. Pour cela, il faut les changer de place dans le lombricomposteur. Il faut également les nettoyer de temps en temps. Le lombricomposteur doit être placé dans un endroit sec et à l’abri de la lumière. Il doit être installé dans un lieu calme et aéré.

Comment savoir si mon lombricomposteur est plein?

Si vous voyez que le lombricomposteur est plein, c’est qu’il est temps de le vider. Vous pouvez le faire tous les 2 mois. Vous pouvez aussi le vider plus souvent si vous constatez que le lombricomposteur est trop rempli.

Comment fabriquer un lombricomposteur

Le lombricomposteur comprend deux parties distinctes : La partie inférieure, celle où les vers vivent, est en plastique et s’intègre parfaitement dans n’importe quel espace. La partie supérieure comprend le bac à compostage, qui sert à stocker les déchets de cuisine qui vont être ainsi transformés. Cette partie est faite de bois afin de lui donner une forme ronde, comme celle d’un seau. Une fois que les vers ont fini de transformer les déchets de cuisine, ils remontent à la surface et expulsent les matières organiques au travers d’un tuyau. Elles sont ensuite stockées dans la partie supérieure, pour être utilisées lors des prochaines transformations. Le compost est le résultat de la transformation des déchets par les vers. Il ressemblera à un terreau frais et vert, et constituera ainsi un bon engrais. Ce compost peut être utilisé comme fertilisant naturel dans le jardin ou sur la terrasse.
